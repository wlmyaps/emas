Shio • Zodiak • Planet • Elemen • Weton

Panduan lengkap dan komprehensif mengenai astrologi, astronomi, dan kalender tradisional. Jelajahi misteri alam semesta dan warisan leluhur dalam satu tempat.

12 Shio Tiongkok

Shio adalah siklus 12 tahun dalam astrologi Tionghoa. Setiap tahun diwakili oleh hewan dan karakteristiknya. Berikut data lengkap semua shio beserta tahun, elemen, sifat, dan keberuntungan.

12 Zodiak Barat

Zodiak Barat membagi lingkaran ekliptika menjadi 12 bagian, masing-masing 30°. Berdasarkan posisi matahari saat kelahiran. Berikut data lengkap semua zodiak.

8 Planet Tata Surya

Tata surya kita terdiri dari 8 planet yang mengelilingi matahari. Berikut data astronomi lengkap untuk setiap planet.

5 Elemen (Wu Xing)

Lima elemen (Wu Xing) adalah konsep fundamental dalam filosofi Tiongkok yang menjelaskan interaksi dan hubungan antar fenomena alam.

Siklus Wu Xing

🔄 Siklus Generatif (Saling Melahirkan)

Kayu → Api → Tanah → Logam → Air → Kayu

  • • Kayu menghidupkan Api (kayu dibakar)
  • • Api menghasilkan Tanah (abu)
  • • Tanah membentuk Logam (mineral)
  • • Logam mengumpulkan Air (kondensasi)
  • • Air menumbuhkan Kayu (irigasi)

⚔️ Siklus Destruktif (Saling Mengalahkan)

Kayu → Tanah → Air → Api → Logam → Kayu

  • • Kayu mengalahkan Tanah (akar menembus tanah)
  • • Tanah menahan Air (bendungan)
  • • Air memadamkan Api
  • • Api melelehkan Logam
  • • Logam menebang Kayu (kapak)

35 Weton Jawa

Weton adalah kombinasi hari (7 hari) dan pasaran (5 hari) dalam kalender Jawa. Total terdapat 35 kombinasi unik (7×5=35). Setiap weton memiliki neptu (nilai) dan watak tersendiri.
